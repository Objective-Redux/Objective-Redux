Class ComponentConnector

Builder that connections a React component to the Objective Redux register, allowing the component to use the states and dispatch events.

This provides the React component with a register prop, which is an instance of the ReduxRegister connected to the components closest provided ancestor. It also provides props from the states that were added.

As an alternative for functional components, the useRegisterFromReactContext hook can be used to get the ReduxRegister.

example
export default ComponentConnector
  .addPropsTo(MyReactComponent)
  .from(MyStateControllerOne)
  .from(MyStateControllerTwo, slice => ({ a: slice.a }))
  .connect();
